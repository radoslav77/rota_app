What are Staffing Rules?

Rules define exactly how many staff from each kitchen section are needed when an event has a certain number of pax. The algorithm uses your rules first; if no rule matches, it uses built-in tier defaults.

Pax Tiers (defaults)

Tasting: 0 pax · Small: 1–30 · Medium: 31–100 · Large: 101–250 · VIP: 251+

Custom rules override these defaults for the matching section + pax range.

Shift Suggestions

The shift time associated with a rule is suggested to the algorithm when it is allocating staff on event days. Common values: 0800-1630, 1200-2100, 1000-1830.
